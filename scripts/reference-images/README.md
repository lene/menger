# Reference Images

This directory contains golden reference images for visual regression testing.

## Purpose

These images represent the expected output of the renderer for various test cases. During integration tests, newly rendered images are compared against these references to detect unintended visual changes.

## Contents

Reference images are automatically generated when running tests with `--update-references`:

```bash
./scripts/integration-tests.sh ./menger-app/target/universal/stage/bin/menger-app --update-references
```

## File Naming Convention

Files are named based on the test case name with special characters replaced by underscores:
- Test: "sphere" → `sphere.png`
- Test: "tesseract (4D hypercube)" → `tesseract__4D_hypercube_.png`

## Version Control

**Important**: These files MUST be committed to git for CI/CD to work correctly.

When updating reference images:
1. Review the visual changes carefully
2. Ensure changes are intentional (new features, bug fixes, etc.)
3. Commit with a descriptive message:
   ```bash
   git add scripts/reference-images/
   git commit -m "test: Update reference images for fractional level rendering"
   ```

## Do Not Manually Edit

Reference images should only be generated by the test scripts. Manual editing may cause tests to pass incorrectly.

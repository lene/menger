# Per-scene caustics-validation thresholds (Sprint 33).
#
# Format:  <scene>  <resolution>  <max_MSE>  <max_FLIP>
#   scene       basename of the .pbrt twin (without extension)
#   resolution  WxH the menger + reference images were produced at
#   max_MSE     imgtool diff --metric MSE upper bound (clipped-linear PFM pair)
#   max_FLIP    imgtool diff --metric FLIP upper bound (mean perceptual error)
#
# C8 gate (locked in Task 33.8):
#   PRIMARY criterion is the caustic-DELTA spatial correlation (Pearson) between the
#   menger and pbrt on−off caustic maps, via compare-caustic-delta.sh:  correlation >= 0.80
#   (canonical achieves 0.859). Whole-image menger-vs-pbrt is NOT a tight gate: menger renders
#   primary-ray caustics only (no SDS paths) and uses a non-physical constant-ambient diffuse
#   term, so the whole-image energy differs from pbrt by ~5x on the caustic and ~30% on the
#   floor. That is a documented structural limitation (docs/BACKLOG.md F-PBR-DIFFUSE /
#   F-CAUSTICS-SDS), not a caustic defect — see docs/caustics/CAUSTICS.md.
#
# The MSE/FLIP columns below are therefore LOOSE whole-image sanity bounds (measured
# canonical: MSE ~0.25, FLIP ~0.65) that only catch gross breakage, not a precision gate.
# FLIP (Andersson et al. 2020, NVIDIA) is imgtool's perceptual metric (replaces SSIM: the
# installed ImageMagick 6 lacks SSIM).

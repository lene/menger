## 6.2 Ray Tracing Flow (Single Ray)

```
1. Camera ray generation
   в””в”Җв–ә Calculate ray origin + direction from pixel coordinates

2. Trace primary ray
   в””в”Җв–ә optixTrace(handle, origin, direction, ...)

3. Intersection test
   в”ңв”Җв–ә Sphere/Cone/Cylinder: Custom IS programs (analytical)
   в”ңв”Җв–ә Menger4D/Sierpinski4D/Hexadecachoron4D: Custom IS + hit (GPU fractal eval)
   в””в”Җв–ә Mesh: Built-in triangle intersection

4. Closest hit evaluation
   в”ңв”Җв–ә Opaque (alpha вүҘ 0.996)
   в”Ӯ   в””в”Җв–ә Diffuse shading + shadow test
   в”Ӯ
   в”ңв”Җв–ә Transparent (alpha вүӨ 0.004)
   в”Ӯ   в””в”Җв–ә Continue ray through object
   в”Ӯ
   в””в”Җв–ә Semi-transparent (glass)
       в”ңв”Җв–ә Calculate Fresnel reflectance
       в”ңв”Җв–ә Trace reflection ray (recursive)
       в”ңв”Җв–ә Trace refraction ray (Snell's law)
       в”ңв”Җв–ә Apply Beer-Lambert absorption
       в””в”Җв–ә Blend reflection + refraction

5. Fog attenuation (if fog_density > 0)
   в””в”Җв–ә `applyFogInPlace()`: exp(-density * t), blend toward fog color

6. Miss (no hit)
   в”ңв”Җв–ә Check plane intersection
   в”Ӯ   в”ңв”Җв–ә Solid color or checkerboard
   в”Ӯ   в””в”Җв–ә Shadow test
   в””в”Җв–ә Return background color

7. Accumulate into pixel buffer
```

## 6.4 Geometry Generation (SpongeBySurface)

```
1. Start with 6 initial faces (cube sides)
   в””в”Җв–ә Face(center, scale, normal)

2. For each level (0 to N):
   в””в”Җв–ә For each face:
       в”ңв”Җв–ә Subdivide into 9 squares
       в”ңв”Җв–ә Remove center square (hole)
       в”ңв”Җв–ә Add 4 rotated faces (inner walls)
       в””в”Җв–ә Result: 12 faces per input face

3. Face count: 6 Г— 12^level
   в””в”Җв–ә Level 0: 6 faces
   в””в”Җв–ә Level 1: 72 faces
   в””в”Җв–ә Level 2: 864 faces
   в””в”Җв–ә Level 3: 10,368 faces

4. Convert to triangles (2 per face)
   в””в”Җв–ә Export vertex + index buffers
```
